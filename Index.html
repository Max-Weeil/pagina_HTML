    Inteligencia

    A partir de la página web, se busca mostrar la importancia del papel que cumplimos los psicólogos a través de la aplicación y evaluación de pruebas para la medición y comprensión de los niveles de inteligencia.

    1. En la página web se incluirán los siguientes apartados, tomando como referencia a Arrimada, M (2021) quien menciona los siguientes tests. 

    GENERALIDADES

    1¿Cómo se puede realizar una evaluación de la inteligencia de una persona?

    No existe un método único para evaluar la inteligencia humana, sino que se han desarrollado varios tipos de pruebas para tal fin, entre las que destacan varias, de las que hablaremos en esta página web. Las pruebas diseñadas para evaluar la inteligencia pueden medir aspectos más globales (como la comprensión del lenguaje), por lo que con la ayuda de estas herramientas, los psicólogos pueden evaluar y comprender el potencial intelectual de una persona entre otras habilidades.     3¿Qué es el coeficiente intelectual?

    - El coeficiente intelectual, o cociente intelectual, es definido por Psynthesis Psicología (2023) como un número que refleja la capacidad de una persona para utilizar el razonamiento lógico y resolver problemas.. Es decir, se utiliza para cuantificar aspectos relacionados con las capacidades cognitivas de un individuo. La mayoría de las personas que realizan una prueba de inteligencia tienen un coeficiente intelectual promedio de 100, por lo que este número se define como inteligencia promedio. Además, cuando se utiliza una prueba para medir el coeficiente intelectual, el desempeño de una persona se mide comparándola con un grupo de personas del mismo grupo de edad. Psynthesis Psicología (2023) menciona que cuando se trata del coeficiente intelectual promedio a diferentes edades, las puntuaciones siempre variarán.
